# tenure-monthlycharges-interaction

📌 Description

This analysis visualizes the relationship between customer tenure, monthly charges, and churn behavior using a heatmap.

🛠️ Steps Performed

Created grouped categories for:

Tenure (0–12, 13–24, etc.)

Monthly Charges (0–20, 21–40, etc.)

Built a Pivot Table:

Rows → Tenure Group

Columns → Charges Group

Values → Count of Churn

Filtered only churned customers

Applied Conditional Formatting (Color Scale) to create heatmap

📊 Key Insights

Customers with low tenure and high monthly charges show highest churn

Long-term customers are more stable

Pricing plays a critical role in retention

💡 Recommendations

Provide early-stage offers for new customers

Reduce pricing pressure on high-paying customers

Improve retention strategies for first-year users
